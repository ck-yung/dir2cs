# Color Feature

## Syntax
```--color COLOR```

```--color INTEGER,COLOR,COLOR-OF-TOTAL-LINE```

```--color INTEGER,COLOR,COLOR-OF-TOTAL-LINE,BACKGROUND-COLOR-OF-TOTAL-LINE```
