dir2 will load the following options from config file
  ~/.local/dir2.opt or %USERPROFILE%/.local/dir2.opt

--regex
--utf8
--case-sensitive
--reverse      CONFIG
--sort         CONFIG
--size-format  CONFIG
--count-format CONFIG
--date-format  CONFIG
--hidden       CONFIG
--excl         CONFIG
--excl-dir     CONFIG
--link         CONFIG
--hide         CONFIG
--show         CONFIG

No shortcut will be referred.

You can apply '--cfg-off' in envir 'dir2' or command line arg
to skip the config file.
